신청인 김용수(21242400691), 접수일자 : 2024.03.03
 
은행권의 상속인 금융거래조회 서비스 결과취합이 완료되었습니다.  
금융감독원 또는 은행연합회 홈페이지에서 직접 확인하시기 바랍니다.  
참고로 금융감독원 홈페이지에서는 각 금융협회의 결과를 일괄로 조회하실 수 있습니다.  
\<조회방법\>  
① 금융감독원 홈페이지 접속 → 상단 메뉴 중 민원·신고 → 상속인조회 → 상속인 금융거래조회 바로가기 → 신청인이름, 신청접수번호 입력 → 휴대전화번호 입력 → 결과조회  
② 은행연합회 홈페이지 접속 → 상단 메뉴 중 소비자포털 → 금융서비스정보 → 상속인금융거래조회 → 조회하기 → 접수번호, 신청인 성명 입력 → 휴대전화 본인인증 → 결과조회  
\<유의사항\>  
접수번호를 모르시면 상속인 금융거래조회 서비스를 신청하신 기관(주민센터 또는 은행 등)에 확인하시기 바랍니다.
