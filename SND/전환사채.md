# 전환사채 (CB, Convertible Bond)

SND가 2026년 5월 중 실행 예정인 자금 조달 수단.

## 현황

- **실행 목표: 2026년 5월 중**
- 계약 체결 전 법무사 사전 검토 필수 — [[법무사 미팅 26-05]] 참조

## 법무 절차

전환사채 실행 시 **정관 변경 + 변경등기** 필요 (법무사 확인 사항):

1. 전환사채 관련 조항 정관에 반영
2. 변경등기 신청

→ 스톡옵션 관련 [[정관 변경]] 등기와 별도로 진행되는 두 번째 등기 절차.

## 개념 메모

전환사채는 일정 조건 충족 시 주식으로 전환 가능한 채권. 비상장 스타트업에서 투자 유치 시 자주 사용.
