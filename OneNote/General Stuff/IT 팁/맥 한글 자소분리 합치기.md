설치 – pip install unicode-nfd2c  
사용법  
현재 작업중인 디렉토리(\.)에서 하위(-r) 디렉토리를 포함해서 분리된 한글 자소 목록을 출력합니다.  
python nfd2c.py -r . (맨 뒤에 쩜 추가)  
현재 작업중인 디렉토리(\.)에서 하위(-r) 디렉토리를 포함해서 분리된 한글 자소를 결합(-x)합니다.  
python nfd2c.py -r -x . (맨 뒤에 쩜 추가)
 
해당 py 파일이 있는 경로를 사용해야 작동한다.  
python C:\Python311\Lib\site-packages\nfd2c\nfd2c.py -r -x .
 
nfd2c 라고 바로 실행해도 된다.  
Command 창 열고 nf2dc -r -x . 해도 된다.
