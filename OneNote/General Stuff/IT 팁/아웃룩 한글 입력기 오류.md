Outlook 2016 사용 중 새 메일 작성 시 문자열 마무리 창이 자동 실행되는 증상으로 문의 주셨습니다.  
빌드 업데이트 이슈일 수 있습니다. 아래와 같이 진행해 보실 것을 안내해 드립니다.  
1. 명령 프롬프트를 관리자 권한으로 실행  
로고 키+ X 키 -\> 명령 프롬프트 관리자 권한으로 실행  
2. CD C:\Program Files\Common Files\microsoft shared\ClickToRun 입력 후 ENTER  
* 복사하여 붙여넣기를 할 수 있습니다.  
3. officec2rclient /update user updatetoversion=16.0.7341.2035 입력 후 Enter  
* 복사하여 붙여넣기를 할 수 있습니다.
