현재 stealth 이미지 - 비번은 도토리원  
**1. 디스크 유틸리티 실행**

- Spotlight (⌘ + Space) → **디스크 유틸리티** 실행
 
**2. 새 이미지 생성**  
상단 메뉴에서  
**파일 → 새로운 이미지 → 빈 이미지…** 선택
 
**3. 이미지 옵션 설정**  
아래 항목을 그대로 설정한다.

|   |   |
|---|---|
|**항목**|**설정값**|
|이름|원하는 이름|
|저장 위치|원하는 위치|
|크기|원하는 최대 크기 (예: 100GB)|
|포맷|**APFS**|
|암호화|**AES-256**|
|이미지 포맷|**Sparse Bundle Image**|
|키체인에 암호 저장|체크 해제|

비밀번호 입력 후 **생성(Create)** 클릭
 
**4. 암호화 폴더 사용**

1. 생성된 .sparsebundle 파일 더블 클릭
2. 비밀번호 입력
3. 열리는 폴더 안에 파일 추가 / 수정 / 삭제
 
**5. 사용 종료**

- Finder 사이드바에서 해당 볼륨 **추출(Eject)**
 
**완료**  
이제 해당 폴더는

- 열 때마다 비밀번호 필요
- 닫히면 자동으로 암호화됨
