하드웨어 변경 후 Windows 10 다시 정품 인증  
[https://support.microsoft.com/ko-kr/help/20530/windows-10-reactivating-after-hardware-change](https://support.microsoft.com/ko-kr/help/20530/windows-10-reactivating-after-hardware-change)
 
Windows 10(버전 1607 이상)에서 Microsoft 계정을 디바이스의 Windows 10 디지털 라이선스에 연결할 수 있습니다. 이렇게 하면 마더보드 교체처럼 나중에 대대적으로 하드웨어를 변경하는 경우 활성화 문제 해결사를 사용하여 Windows를 다시 정품 인증하는 데 도움이 될 수 있습니다.
 
PC에서 실행 중인 Windows 10 버전을 확인하려면 시작 단추를 선택한 다음 설정 \> 시스템 \> 정보를 선택합니다.
 
Microsoft 계정을 추가하고 디지털 라이선스에 연결하려면  
1.시작 단추를 선택하고 설정 \> 업데이트 및 보안 \> 정품 인증 \> 계정 추가를 차례로 선택합니다. Microsoft 계정을 추가하려면 관리자로 로그인해야 합니다.  
2.Microsoft 계정 및 암호를 입력하고 로그인을 선택합니다. 또한 입력한 Microsoft 계정이 연결된 계정이 아닌 경우 로컬 계정의 암호를 입력해야 합니다.  
3.Microsoft 계정을 추가한 후에는 정품 인증 페이지에 Windows가 Microsoft 계정에 연결된 디지털 라이선스를 사용하여 정품 인증되었습니다.가 표시됩니다.
 
디바이스에 이미 Microsoft 계정을 추가한 경우 정품 인증 페이지에 Windows가 Microsoft 계정에 연결된 디지털 라이선스를 사용하여 정품 인증되었습니다.가 표시됩니다.
 
**대대적인** **하드웨어 변경 후 정품 인증 문제 해결사 사용**
 
Microsoft 계정을 추가하고 디지털 라이선스에 연결한 경우에는 대대적인 하드웨어 변경 후 정품 인증 문제 해결사를 사용하여 Windows를 다시 활성화할 수 있습니다.
 
Microsoft 계정을 추가하고 디바이스의 디지털 라이선스에 연결하지 않은 경우 하드웨어 변경 후에 정품 인증 문제 해결사를 사용하여 Windows를 다시 활성화할 수 없습니다.
 
1.시작  단추를 선택하고 설정  \> 업데이트 및 보안  \> 정품 인증  \> 문제 해결을 차례로 선택합니다. 관리자로 로그인해야 합니다.  
2.문제 해결사에는 디바이스에서 Windows 정품 인증을 할 수 없다는 메시지가 표시됩니다. 최근에 이 디바이스의 하드웨어를 변경했습니다.를 선택하고 다음을 선택합니다.  
3.Microsoft 계정 및 암호를 입력하고 로그인을 선택합니다. 또한 입력한 Microsoft 계정이 연결된 계정이 아닌 경우 로컬 계정의 암호를 입력해야 합니다.  
4.Microsoft 계정에 연결된 디바이스 목록에서 현재 사용 중인 디바이스를 선택하고 지금 사용 중인 디바이스임 옆의 확인란을 선택하고 정품 인증을 선택합니다.
 
결과 목록에 사용 중인 디바이스가 표시되지 않으면 디바이스에서 Windows 10 디지털 라이선스에 연결한 동일한 Microsoft 계정을 사용하여 로그인되어 있는지 확인합니다.  
올바른 Microsoft 계정을 사용하여 로그인된 경우 다음과 같은 몇 가지 추가 이유 때문에 Windows를 다시 정품 인증하지 못할 수 있습니다.
 
•디바이스의 Windows 버전이 디지털 라이선스에 연결한 Windows 에디션과 일치하지 않습니다.  
•정품 인증하는 디바이스 유형이 디지털 라이선스에 연결한 디바이스 유형과 일치하지 않습니다.  
•디바이스에서 Windows가 정품 인증된 적이 없습니다.  
•디바이스에서 Windows를 다시 정품 인증할 수 있는 횟수 제한에 도달했습니다.  
•디바이스에 둘 이상의 관리자가 있으며 다른 관리자가 이미 디바이스에서 Windows를 다시 정품 인증했습니다.  
•디바이스가 조직에서 관리되고 있고 Windows를 다시 정품 인증하는 옵션을 사용할 수 없습니다. 정품 인증에 대해 도움이 필요한 경우 조직의 지원 담당자에게 문의하세요.  
디바이스의 Windows를 다시 정품 인증하기 위한 추가 도움이 필요한 경우 고객 지원 서비스에 문의하세요.
